0. 구글 파이어베이스에 가서 Hosting : 시작하기 
1. vsc에서 아래 내용을 수행
   npm install -g firebase-tools@latest (컴퓨터마다 한번만)
   firebase login (상황에 따라 logout되면 login을 하면 됨)
       Allow Firebase to collect CLI and Emulator Suite usage and error reporting information? No
       Success! Logged in as chamsogum@gmail.com
   firebase init (프로젝트마다 한번만)
      "Are you ready to proceed?" Y
      (*) "Hosting: Configure files for Firebase Hosting and (optionally) set up GitHub Action deploys"
      "What do you want to use as your public directory?"(public) dist 입력하고 엔터
      "Configure as a single-page app (rewrite all urls to /index.html)?" N
      "Set up automatic builds and deploys with GitHub?" N
      "File dist/index.html already exists. Overwrite?" N
   
2. https://ko.vitejs.dev/guide/static-deploy (정적 웹 페이지로 배포하기 : Google Firebase)
   프로젝트 루트 폴더에 두 개 파일 작성 : firebase.json, .firebaserc (프로젝트 ID 수정) 

3. vsc : npm run build => 프로젝트 폴더에 dist 폴더가 생성됨
4. vsc : firebase deploy
5. 카카오맵 Web 플랫폼에 파이어베이스 호스팅 주소 추가
6. 코드가 수정될때마다 3, 4번만 반복
